## What I Do

> **I build products that push the boundaries of AI** — whether it's an autonomous pentester that finds vulnerabilities before hackers do, an adaptive AI tutor that personalizes learning for every student, or a Web3 tool that brings blockchain data to your fingertips. If it involves intelligence, I can build it.

- **AI & LLM Systems** — RAG pipelines, autonomous agents, multi-model workflows, and chatbots
- **AI Security** — Autonomous pentesting, OWASP coverage, proof-of-concept exploitation
- **Full-Stack Web** — React, Next.js, FastAPI, Node.js with PostgreSQL, Supabase, and MongoDB backends
- **Web3 & Blockchain** — Solidity smart contracts, Ethereum dApps, on-chain data tools
- **Mobile Apps** — React Native and Flutter cross-platform development
- **DevOps** — Docker, Vercel, CI/CD pipelines, Linux server management
